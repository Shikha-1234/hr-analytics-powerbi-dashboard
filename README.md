# IBM HR Analytics Dashboard



## Overview



This Power BI project analyzes employee attrition and workforce trends using the IBM HR Analytics Employee Attrition \& Performance dataset.



The dashboard focuses on understanding employee turnover patterns across departments, job roles, demographics, job satisfaction levels, and overtime status. The goal is to identify factors associated with attrition and provide insights that can support workforce planning and retention strategies.



---



## Dataset Information



* Dataset: IBM HR Analytics Employee Attrition \& Performance

* Total Employees: 1,470

* Total Columns: 35

* Tool Used: Power BI Desktop
